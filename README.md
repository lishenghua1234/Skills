# 🤖 SearchNews: Agentic XHS Material Generator

这是一个由 AI 驱动的全自动“科技新闻搜集 -> 小红书爆款图文创作 -> 智能生图”引擎。
最初作为知识类短视频的剧本拆解器，但经过高度提纯重构后，目前它是专为 **AI 与前沿科技领域自媒体** 打造的无人值守素材工厂，可以直接与 [OpenClaw](https://github.com/openclaw/openclaw) 等自动化框架结合组成终极的零干预发布流水线。

## ✨ 核心特性

- **📡 自动嗅探与聚类**：内置全球顶级科技媒体的 RSS 源获取（默认配置涵盖 MIT Tech Review, TechCrunch, The Verge, HuggingFace 日报等），自动获取过去指定小时内的新闻，并通过余弦相似度算法智能合并相同事件。
- **🧠 爆款图文改写 (Gemini 驱动)**：抛弃干瘪的机翻，内置了独家的“小红书网感 Prompt”，由大模型接管所有事件重写。它会提取极具吸引力的主旨标题、富含 Emoji 表情符号的价值论述、精简扼要的知识要点，让生成后的排版开箱即发。
- **🎨 视觉生成推荐**：对于每篇解析完毕的图文，它不仅生成文案，还能自动设计“视频感/网感配图指示”，进而利用 Imagen API 输出适配竖屏 (3:4) 的绝佳配图。
- **🛡️ 军用级 API 容灾轮询池**：不用再被 429 和配额困局打断工作。在 `sources.yaml` 中你可以配置无数个白嫖来的/不同账号的 Gemini 或 Minimax Key 组合成 `api_keys` 列阵，遇到网络限滑或配额抽风，底层 `KeysManager` 会智能熔断并无缝切换至下一把备用秘钥继续干活。
- **📦 标准化输出格式**：专为 Agent (如 OpenClaw) 喂饭而生。执行一遍，它就会在你的文件夹系统下创建若干以“吸引眼球标题+短码”命名的图文包目录，里面自动放好了规整的 `material.json` 数据及其附送的一整套顺序排列的生图。

## 🚀 快速开始

本项目依赖 `uv` 进行现代化的 Python 环境与依赖调度。请确保你的机器已经安装了 `uv` 工具。

### 1. 克隆与配置环境变量

首先抓下这份代码。然后进行至关重要的配置部署。
我们已经去除了所有需要在命令行终端设定的 `$env`，所有的配置仅需要一个文件集中管理。

```bash
# 拷贝出你自己的私密配置文件（已被 Git 忽略处理）
cp sources.example.yaml sources.yaml
```

打开刚刚创建好的 `sources.yaml`，在最下方的 `api_keys` 选项组中填入你申请到的真正大模型秘钥：
```yaml
api_keys:
  gemini:
    - "AIzaSy_YOUR_GEMINI_KEY_1"
    - "AIzaSy_YOUR_GEMINI_KEY_2" # 可以放无数个，会自动切换
```

### 2. 执行生成！
整个世界极其简单，调用以下一行指令，项目便会根据源列表扫描最新的新闻并开始自动写作与作图。

```bash
uv run python -m src.cli run --since 12h --max-items 3
```

- `--since 12h`: 扫描过去十二小时内发出的新闻动态来处理。
- `--max-items 3`: 即使最近发生了上百条新闻，只优选含金量最高的 3 条做成发帖素材。

当命令行跑完（大概需时一分钟），你会发现你的项目根目录里多出了一个名为 `素材/` 的神奇文件体系。里面已经躺着可以立刻一键多图加上段落发上小红书的高级笔记装了。

## 🔌 联级到你的 Agent 机器人系统 (OpenClaw)

这也是本引擎的真正用途。如果你同时也是诸如 **OpenClaw** 大模型控制机器人的拥趸，你可以利用我们提供的 `SKILL.md` (或通过查阅里面的指引) 直接给你的机器人加装图鉴。
不需要手搓任何脚本，在把本文件夹放入正确路径后，只需在微信/QQ等地方给你的机器人助手发信息：
> “*去收集最近 5 小时的科技新闻并在小红书发布。*”

机器人便懂得把 `5` 抽取成参数交给此项目底层执行生成，等生成完毕后自动在 `素材` 目录下摘取 Json 段落和自动绘图去完成无人直播式的霸屏发布！💯
